# 💰 SpendWise - Personal Finance Manager

A full-stack web application for managing personal finances.

## 🚀 Features
- ✅ Secure Login & Registration
- ✅ BCrypt Password Security
- ✅ Expense Tracking by Category
- ✅ Monthly Budget Alerts
- ✅ PDF Report Download
- ✅ Forgot Password with OTP
- ✅ Annual Overview Dashboard
- ✅ Interactive Charts

## 🛠️ Tech Stack
- Java Servlets & JSP
- MySQL Database
- Apache Tomcat 11
- BCrypt Security
- JavaMail API
- iText PDF
